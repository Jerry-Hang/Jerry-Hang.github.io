关于我是如何接触电子产品到智能终端，再到现在。更多的是个人往事，方便后来的开发者了解我是一个怎么样的人，技术含量不是很多，就当一个乐子看吧。

## 我的过去

我是 Jerry，Huang（Hang 是在 GitHub 命名时发现被占改的）是我的姓氏。我的祖籍是江西，父母是从外地来深的纯正客家人。我三岁时来到深圳上幼儿园，直到小学五年级才有一部手机（虽然很快就没有了）。

我的家庭环境算小康但不是中产，生活水平跟很多来深工作者一样。父母的思想算是相对开放的，但是对电子产品的管控在初中之前一直都是很严的，导致我在前期就像山顶洞人一样，获得的信息非常滞后，但这也成为了我日后的动力来源。

我接触的第一个游戏是《我的世界》，当时应该是 2016 年还是 2017 年吧，我刚上小学。从那之后就再也没玩过其他游戏，直到初二才接触到我的第二款游戏《暗区突围》（虽说现在已经退游了）。

刚上初二没多久，我用上了我妈闲置的办公本（i5-1135G7，8+512GB），那是我第一次接触 Windows，也是我第一次认清什么是生态——即使是最弱的 x86 生产力，也根本不是 ARM 架构能够轻易挑战的。再往下就是后话了。

在初期我比较喜欢听一些比较二的歌（没错，就是那种听起来很燃而且有歌词的那种，现在我也不怎么喜欢听了），到现在我听得更多的是后朋克音乐和纯音乐，以悲伤和平缓的调子为主，主要在我做事的时候放，平复躁动的心……

现在的游戏呢？我也不知道自己在玩什么，主要是以 3A 为主吧。体验过《死亡搁浅》，玩过《2077》，在西部体验了摩根的传奇经历，还有《底特律》《识质存在》……现在唯一偶尔会登一下号、刷刷材料的东西，可能就是崩铁了（米家的其他两款游戏我也玩过，但要么是新手教程还没做完就退了，要么是刚过完新手教程主线还没怎么做就退了）。

## 对编程与 AI 的看法

如果从涉及的技术以及常用的工具、编程语言来说的话，我应该算 3/4 个全栈开发者。我目前最常用的编程语言是 Python，其次就是 C 语言与 C++，现在正在学习 Java。这些语言呢，其实大部分都处于三脚猫功夫或者半桶水的状态。我编程的时候并不会亲力亲为地把代码全部写完，因为我觉得太蠢了。在现在 AI 编程如此成熟的情况下，这种很传统的手艺人工作，我觉得跟 AI 比已经没有优势了——我不是说这样做不如 AI，我是说这样做效率真的有点低了。但我还是很敬佩在这种环境下仍然能够坚持纯手写代码的人。

我之所以敢用 AI，可能是因为逻辑和代码直觉吧。只要是 AI 能完成的简单模块化任务，我就不会手写；只要是不涉及创新以及过度复杂的东西，AI 能干的，我绝不插手。

我知道我这样做可能会被某些人鄙视，但我表示理解。现在写程序的无非就三类人：

一类是像我这种比较中性的，自己对代码的理解不是很深，但会善用工具。

还有一类非常鄙视 AI，觉得 AI 写的都是他妈的垃圾，一定要纯手写的才是真正能用的、可以信任的、可以推广的，面对 AI 创作出来的代码产品，一律觉得非常恶心或嗤之以鼻。我不理解为什么要这样？就像在工业革命的前夕，明明已经知道大势所向，一位织布工人不是去学习如何驾驭工业化织布机，而是还在那里用纯手工的个人织布机织布，在我看来无非是极其愚蠢且傲慢的。人生来就是使用工具的，不是跟工具比在单一长处上谁效率更高的。

另一类就是什么都不懂、什么都不会、什么都不学，然后有一个 AI 就觉得自己什么都能干了。结果就是很典型的一个英文报错什么都看不懂，一个压缩包都不知道怎么解（你别笑，还真有这种神人），还有搞不清什么是 C 语言与 C++ 的。报错是从来不看，代码是不会审查，生态是不会学习，甚至他妈的出了问题都不知道把出错的地方截个图或者口述转述给 AI，告诉它错在了哪里。这种典型的巨婴思维，就算是两位老祖到他身边都没用。工具为什么叫工具？因为它是一个不会自己思考的东西，而驾驭工具的人要先学会如何使用工具。这种人就是他妈的把两位老祖丢他身边、手把手去教都教不会的那种傻逼，这边建议先从科普读物读起，再过来理解怎么用 AI 吧。

AI 为什么是生产力工具？是因为它把人类从繁重、复杂且无意义的纠错，以及常规片段式代码写作任务中解放了出来，可以让人类工作者专心致志地用于创新、思维、发散等其他方面。但这并不意味着人类作为工具使用者可以在什么都不懂的情况下使用工具。相反，这反倒需要使用者具有架构与生态的概念、代码的逻辑性、直觉性，以及基础的技术概念。还有，不要指望工具能够代替一切，当工具代替一切的时候，你就没有意义了！

当然，这些我个人的感悟或许措辞过于激烈了一些，但这确实是我的真实想法。而今天这些经历，还得从我人生接触的第一个终端产品说起。

## 初遇编程

那个时候应该是我三四年级的样子吧，深圳当时是由官方还是哪个品牌——我也不太清楚——赞助了一个叫“寻星计划”的项目（一个包含了航天探索与编程的公益比赛项目）。当时的我通过了初赛，之后成功进入了复赛阶段，那是一个在坪山区举行、为期两天的比赛（参赛选手要过夜，并且无父母陪同）。

那是我第一次接触到图形化积木编程以及开发板，当时的我几乎是零基础去学的。题目是让我们以“太空小镇”为主题，搭建一个具有未来科技化的小镇。因为几乎不懂编程，我做的小镇比较简陋——概念上是一个人造太阳（用一个环形灯来代替），可以通过调节颜色、亮度以及通过舵机来控制方位的演示灯；然后又用超声波传感器做了一个简易的 2D 平面雷达（我当时好像是把它当作这个小镇的预警系统吧）；还有几个其他的功能，我也记不清了。

我只记得当时我是唯一一个按照题目要求做完题目的人（我知道这很抽象，但确实有一大堆人没达到这个要求）。后来就稀里糊涂地拿了奖，然后呢，上了电视，好像在央视几频道播出过——呃，我也不太记得了，好像是直播。那是我最直观地感受到编程是什么概念。

## 从华为畅享 10 到 iPad 的启蒙

疫情开始之后，我有了第一台手机，那是一台颇有年龄的华为畅享 10，用着麒麟 710F 芯片。说实在的，初代鸿蒙直接把我毒打了一顿（而且发热居然还不小，直接给我带来了小小的一波非高通的火龙震撼）。可能是我心理建设不足吧，我实在是难以想象——虽然说我妈只用了不到三年左右吧——我只是没想到还能卡成这副模样。那一次是我从实际体验上感受到了什么是生态，什么是系统，什么是芯片……

后来因为一些原因，这个手机我用了不到一个月，我父母就收了回去。再后来直到初二，我因为一点点人际关系还有一些事情，父母拿了一个年龄比华为畅享 10 还要老的 iPad mini 2（1+32GB）给我。怎么说呢？确实是放开了，有总比没有好。

不过，相比于鸿蒙带给我的震撼，iOS 的震撼对我来说更大。首先就是它那个极其优秀的墓碑机制——我居然发现它并不比华为畅享 10 卡，甚至在固定使用几个软件的情况下，你几乎感受不到卡顿，响应速度居然还比 4GB 内存的畅享 10 要流畅一些……但是，受限于当时的硬件情况以及我的个人心理状态，那主要是一段很漫长的科普时期，我几乎没有拿它来做过任何编程的事情，更多的是了解什么是硬件，以及有关于硬件和编程的概念。

## 颇有坎坷的电子越狱与树莓派 5 的救赎

那个时候刚好是初二放寒假的一段时间，在回老家过年之前，我偶然间整理家务，发现了一台非常老旧、年纪比我大得多的联想笔记本电脑。因为配置很差，而且年龄很大，父母没怎么管，就把那台电脑给了我，那算是我第一台个人 PC 吧。配置真的不用抱太大幻想，它的年龄甚至比我还要大——处理器是堪称远古时期的 45 纳米光刻工艺的 Intel Core 2 Duo，和一个速度甚至都没有我现在电脑 C 盘快的 DDR2 内存，而且还只有 2GB！装了一个 Windows 7 家庭版。

那一段时间用这个笔记本电脑，简直就是痛苦至极，比用华为手机的时候还要痛苦。用了不到两个多月吧，发现没有任何抢救的希望，这台电脑就被我拆掉了，但那是我第一次接触到了 Windows 的生态以及 x86 架构。

转眼就到了 2023 年末，树莓派 5 首发，终于让犹豫不决的我决定花一千多块钱的预算去买下一整套开发套装。在我的个人存款里面扣掉一千块钱之后，终于得到了我心心念念已久的树莓派，一个真正属于我个人的电子产品。

此后就是长达半年多的密集折腾期。首先就是改装——因为那一套开发套装配套了一块 14 寸的屏幕，我用外壳固定好之后，把它做成了一个半移动式的 Linux 平板。我开始尝试在上面安装程序以及软件，虽说后面屡屡受挫，但我也成功在一个树莓派上面安装上了哔哩哔哩、我的世界，以及一个谷歌浏览器。

但我是一个很难闲得住的人。我花了将近一个多月的时间去 B 站上搜教程，制作系统盘，改 UEFI 引导，补全驱动，解决生态与架构兼容性，终于在连续两天累死累活的折腾下，在这一个树莓派 5 的开发板上跑上了 Windows 11 系统！不过代价呢？网卡没有驱动，显卡没有驱动，网口也没有驱动……就是一个很残废的状态。不过得益于 8GB 的“超大容量”，它的体验比那一台十年前的联想笔记本电脑不知道好到哪里去了。后来玩了几天没意思，就又把它刷掉了。但悲催的是，因为我个人的用法不对，导致它的第一个 mini HDMI 因为插拔次数过多，损坏了。

在刷掉的第二天，我就开始赶制起一个大工程：我想要在上面刷一个安卓 14 的操作系统，把它当手机来用。我当时甚至还计划去买一个 UPS 电源控制器，给它装个电池，配合那 14 寸的 1080p 60 赫兹的触控显示器，做一个平板电脑。

又是一个多月的折磨，我承认我确实很享受这个过程——先是解决识别问题，然后搞定 boot loader，然后刷写驱动，网卡、声卡、显卡等等等等；然后解决存储空间只能识别到 8GB 的问题；然后再通过刷机和 USB 调试解决地区时间校正不准、获得谷歌的安全认证、解决无法访问互联网的问题，再解决程序安装的问题……

终于在这一大堆问题解决好之后，我的第二个 mini HDMI 也坏了。后来逼得我花了两三天的时间学了一下 SSH 远程连接、投屏等其他方式，也终于在上面装上了哔哩哔哩，装上了暗区突围，装上了微信，装上了 QQ，还有崩坏星穹铁道……

可惜啊，当时我并不知道它坏得有多严重。但是又因为一些学业问题和人际关系，我终于说服我父母，让我花自己的钱去装一台自己的台式电脑……

## 在华强北的坑里，窥见数字空间的自由

在装台式机之前，我其实一直用的都是我妈闲置的办公本，那是一台搭载 i5-1135G7 的戴尔办公本。也是这台电脑，让我接触到了人生首个 3A 大作——《底特律：化身为人》，以及玩上了小时候天天看那些主播玩的《深海迷航》。

也是这台电脑，帮助我制作专用于树莓派的 Win 11 系统镜像、修改 UEFI 引导、远程 SSH 联机。我也是在这台电脑上面下载并注册了我的 Steam，正式接触了 Python 编程，以及初步学习如何用 C 语言写代码……

当然，也是这长达快半年的使用体验，更加深刻地让我感受到了什么是架构、什么是生态——即便是在 x86 平台上看似非常弱的处理器，在代码编写以及生产力方面，也不是 ARM 平台能够随便碰瓷的（注意，我说的是“随便”，不是说不能，M 系列就是一个非常典型的例子）。

再然后，就是我说服父母，让他们同意我花自己的钱装电脑了。

当时的我预算很紧张，只有大概 1500 块左右。再加上我并没有很深入地去了解装机所需要注意的事项，被一些营销号给带偏了，就踩了几乎所有垃圾佬必踩的一个坑——我玩上了 X99 平台，对，就是以核心多且以“i9 级”著称的洋垃圾 E5！根本不知道高主频和单核心性能的重要性，更没去在意 IPC 性能，而是盲目追随多核心，选了一个单核心真的非常差的处理器 E5-2680 V4。

而且装这台电脑我去的还是华强北……debuff 直接叠满了。好在老板还算比较良心，也没坑我多少，最后花了一千二还是一千三，E5-2680 V4 + GTX 1060 5GB + 16GB DDR4 ECC 内存条 + 科脑 X99 主板 + 一块 256GB NVMe 朗科固态硬盘。

也是这台电脑，给了我相对于以往来说更加密集、更加疯狂、更加折腾的探索时期。有了这台电脑之后，我先是爽玩了很多以前一直想玩的游戏，从在 VM 虚拟机里部署 Windows 98 到 Windows 11，再到如何搭建《我的世界》服务器、花上一两天时间解决内网穿透，再到学会用 Ollama 部署本地大语言模型……

那是我最快乐的一段时光，因为我懂得很少，没有那么严重的完美主义洁癖，而且那也是我第一次体会到什么是隐私，什么是个人数字空间。

后来用了快一年，了解的东西多了之后，在体验上自然是越来越难受了。后来到了初三，还有一年就要中考了嘛，我父母就把我电脑管得非常严。

我想着反正也不打游戏了，那就再折腾一下吧。于是，我偷偷让班里同学用闲鱼帮我把那颗 2680 V4 卖掉了，然后买了一颗 2667 V3。买回来之后呢，我也去 B 站上搜教程，搞了几个大佬制作的鸡血工具。结果当时的我对超频这一方面的理解还是太少了，上来就给这颗 22 纳米的老同志处理器降压 175 毫伏，然后不出意外地很不稳定，经常蓝屏。好巧不巧的是，我还没有保存原始的 BIOS 设置。等我赶紧把它调回原先负 30 毫伏的设置时，发现还是出问题了——在刷 BIOS 的时候，电脑还处于不稳定状态，结果刷到一半突然重启了，BIOS 直接当场暴毙。

那段时间我是真的挺难受的，根本不敢用这台电脑。后来想着破罐子破摔吧，反正这个老傻子 E5 我也用不下去了，索性联系同学，把他们都卖了。

然后呢，我父母当时把那台华为畅享 10 给藏起来了，以为我找不到，但实际上我找到了，而且还破解了密码。一个人大半夜在那儿借同学的协议号跟商家周旋，搞了大概大半个月的样子，终于把那些硬件都卖出去了。再加上那个时候我中午不在学校吃，是出去吃的，我就每周不吃饭，省下来一百块钱，连续搞了快一个半月的样子。终于是花三百块还是两百九十五买了一个 12100F，剩下的钱买了个 B760 主板。

结果呢，贪了小便宜，买回来的这个主板主控芯片发霉了。后来我爸生气了，我也老实了，自己再一次花钱买了个 H610 主板。我爸非常生气，没收了我的电脑（物理级没收，直接把我装了系统的那块朗科 256GB C 盘拆走藏起来了）。

但是我当时还没老实——华为畅享 10 不是又到我手里了吗？又拿它来玩游戏。不过那东西是真坐牢啊。但当时还是靠这台手机玩《三角洲行动》的全面战场，花了半个多月的时间，打到了将军五段位。

然后那一段时间嘛，晚睡——6 点就要起床，又非要玩到凌晨一两点，每天只睡四个小时。再加上有一次手机没藏好，我父母来我房间打扫卫生的时候被翻出来了。但是那个时候呢，因为快临近中考了，再加上我个人的心理状态吧，我爸就没有很发火，反倒是心平气和地跟我谈了一次，然后我就老老实实去学习了。

后来呢，从一模还是二模的三百多分，到最后中考花了两个多月的时间提分，提到了 427 分，直接给我干力竭了。

然后就是暑假两个多月的不思进取——唉，疯完就啥都玩。荒废了两个多月，啥都玩：玩《荒野大镖客》，玩《我的世界》继续搞内网穿透、继续部署服务器，顺便继续打一下《三角洲行动》，搞了个“天才少年”，还拿了个 S4 赛季的战场统帅；崩铁顺手把主线全部推完了～

## 新平台与新的领悟

再往后就是暑假结束开学了。因为我学的专业是大数据应用与网络信息安全，所以顺理成章地把台式电脑带了过去。玩了大概不到一个月的样子，我的手又痒了。

于是我就学着怎么一个人注册支付宝、绑定闲鱼，搞各种信息与实名认证，自己一个人动手把整台电脑的每个配件单独挂上去卖了。终于在军训结束之后两个多月的样子，凑够了钱买了一个 R5-9600X。当时我都已经开始准备筹划买主板和内存条了，然后就是内存大涨价。因为远超预算的装机成本，我只能放弃这次计划。为了及时止损，只能再把 9600X 卖出去。

本来打算就算了，再搞一套洋垃圾将就着用吧，反正钱也不算少。弄了一颗 1660 V4，显卡的话，贪图那一点性价比，选了一张 Vega 56，结果等所有配件都到齐，开始用 Win PE 装系统的时候，第三次开机，这张矿卡就烧了。

于是我放弃了，彻彻底底地放弃了。我非常老实地在这个学期结束之后向我父母坦白了一切。这件事跟我父母说完之后，他们也没有生气或者怎样，或许是脱敏了，又或许是互相理解吧。他们让我去打工，说既然这么想要一台电脑，那就自己去挣。

转眼间，打工的日子很快就过去了。加上过年收的红包钱、一个多月的工资，还有父母资助的九百块，总算买了一台像样的性能本，让我高兴了好一阵子。

有了它之后，干了很多以前看得到但玩不上的游戏，也终于能跑跑分炫耀一下了，顺便体验了一下 R9 的多核心魅力。还折腾了一下以前在洋垃圾平台上搞过的纯 CPU 推理，以及靠 5060 Laptop 本地部署 7B 的大语言模型，自己动手做了一个 8 位的简易 RISC-V 处理器……

但紧接着，更大的空虚来了。因为我突然间发现，自己的目标似乎太过短浅了，全都集中在所谓的吃喝玩乐上。在那段快乐的时间过去之后，我整个人变得很空虚，因为我不知道自己要干什么，似乎连游戏都失去了味道。

直到后来，我在逛 DeepSeek 的 API 文档和贡献者页面的时候，看到了一个很年轻的开发者 Eric。我点进了他的 GitHub 页面，看了他的个人主页介绍和他的个人博客……

那是一次精神层面上的震撼。一位年龄仅比我大一岁的高二学生，居然能在兴趣爱好方面做到如此有深度。我瞬间感到羞耻、嫉妒、惊讶，百感交集，真正意义上体会到了什么叫百感交集。

于是便有了我的 GitHub 账户，以及这个个人博客网站。

---

到最后我也不知道该说什么，或许人生就是这样吧。当你知道自己为什么而奋斗、为什么而忙碌时，即使一无所获，也会为忙在其中而感到快乐——因为有了目标。
